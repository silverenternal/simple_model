# Dynamic Governance Case Study

This fixture is a local, reproducible project that exercises dynamic code
governance without network access or global configuration changes. It contains
runtime routes, DI registration, an event subscription, env-gated loading,
generated code, and monkey-patch risk.

Run:

```bash
bash examples/dynamic-case-study/run.sh
```

The flow emits dynamic surfaces, a runtime probe plan and observations, merged
dynamic evidence, adoption report, benchmark scorecard, policy report, and macro
simulation.
